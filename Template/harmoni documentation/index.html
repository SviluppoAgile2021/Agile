Getting Started

Harmoni Event Management - is a html/css template based on Bootstrap 4 framework. It allows to built impressive, better & faster, mobile friendly websites.

HTML Structure

This is the base of each HTML file:


                <!-- header - start
                ============================================= -->
                <header id="header-id" class="header-class clearfix">
                ...
                </header>
                <!-- header - end
                ============================================= -->


                <!-- section - start
                ============================================= -->
                <section id="section-id" class="section-class clearfix">
                ...
                </section>
                <!-- section - end
                ============================================= -->

                
                <!-- footer - start
                ============================================= -->
                <footer id="footer-id" class="footer-class clearfix">
                ...
                </footer>
                <!-- footer - end
                ============================================= -->

Grid System

Harmoni Event Management Template supporst all features of Bootstrap Grid System - go to Bootstrap 4 pages and check how does it work.
